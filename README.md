**pyWallPaper可以从Bing接口(默认)获取图片并设为壁纸**

# 用法
- 先运行 init.py 安装第三方库
- 通过 config.toml 自定义源(自带 Bing 源)
- 运行 main.py 获取最新壁纸并设为桌面
- 运行 run_schedule.pyw 定时设置桌面

# 优点
- 支持自定义获取源
- 提供定时支持
